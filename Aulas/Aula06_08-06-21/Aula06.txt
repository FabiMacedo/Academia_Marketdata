O que é Banco de Dados?
Banco de Dados é um local para armazenar dados.
Banco de Dados: conjunto organizado de dados, armazenados num meio físico para tratatmento posterior; base de dados (Michaelis, dicionário)

Analógico

Banco de Dados é uma estrutura computacional compartilhada e integrada que armazena um conjunto de dados(fatos brutos) e seus metadados.

Metadados -> Dados sobre dados 
Trazem dados sobre os dados ex: tabela de alunos Marketdata
